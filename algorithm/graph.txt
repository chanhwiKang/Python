Tree - 순환성이 없는 무방향 그래프
특징
1. 특정하지 않는 한 어떤 노드든지 루트가 될 수 있다
2. 가장 바깥쪽 노드는 리프노드(leaf ndoe)
3. node A에서 node B로 가는 경로는 항상 있으며, 유일하다
4. 노드 개수 = 간선 개수 + 1

자료구조에서의 Tree
특징
1. 부모 -> 자식 관계가 있는 방향 그래프
2. 루트는 하나다

그래프를 코드로 나타내는 방법
1. 인접행렬 (2차원 배열로 구현)
2. 인접리스트 (list로 구현)
인접행렬 vs 인접리스트
1. 인접 행렬은 노드의 제곱만큼 메모리를 차지
2. 인접 행렬은 임의 접근을 이용해 빠르다 (인접행렬 O(1), 인접리스트 O(N))
대부분의 경우 어느쪽을 사용해도 괜찮다

DFS(Depth First Search) - 깊이 우선 탐색
스택이나 재귀를 사용해서 구현한다 일반적으로 재귀를 사용
